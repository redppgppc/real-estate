네이버 부동산 크롤링 서비스 사용 설명서

빠른 시작 가이드

1. 시스템 요구사항

  • Python: 3.8 이상
  • 브라우저: Chrome, Firefox, Edge 등 최신 브라우저
  • 인터넷: 안정적인 인터넷 연결
  • 필수 라이브러리: requirements.txt에 정의된 패키지

2. 설치 방법

1) 프로젝트 복사

git clone <저장소 URL>
cd real-estate

2) 가상환경 생성 (권장)

python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

3) 패키지 설치

pip install -r requirements.txt

4) ChromeDriver 설치 (Selenium 사용 시)

Chrome 브라우저가 설치되어 있어야 합니다. ChromeDriver는 자동으로 설치됩니다.

3. 실행 방법

1) 서버 시작

uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

2) 브라우저에서 접속

http://localhost:8000

3) API 문서 확인

http://localhost:8000/docs

4. 사용법 요약

🗺️ 검색 기능

  1. 홈페이지에서 검색할 지역을 입력합니다 (예: "강남구", "마포구")
  2. "+" 버튼을 클릭하여 여러 지역을 추가할 수 있습니다
  3. 지도를 클릭하여 위치를 선택할 수도 있습니다
  4. 필터를 설정하여 가격대, 면적, 매물 유형을 제한할 수 있습니다
  5. "검색 시작" 버튼을 클릭합니다

📊 지도 조작

  • 확대/축소: 마우스 휠 또는 +/- 버튼
  • 이동: 마우스 드래그
  • 마커 정보: 마커 클릭 시 상세 정보 표시
  • 여러 지역: 여러 지역을 선택하면 모든 지역의 매물이 표시됩니다

💾 데이터 내보내기

  1. 검색 결과가 표시된 상태에서 "내보내기" 버튼을 클릭합니다
  2. 내보내기 형식을 선택합니다 (CSV 또는 Excel)
  3. 파일이 자동으로 다운로드됩니다

🔔 알림 설정

  1. 브라우저 알림 권한을 허용합니다
  2. 특정 조건(가격 변동, 새 매물 등)을 설정합니다
  3. 조건 충족 시 브라우저 알림이 발송됩니다

5. 주요 기능 설명

다중 지역 검색

여러 지역을 동시에 검색하여 비교할 수 있습니다. 각 지역의 매물 정보를 한눈에 파악하세요.

실시간 데이터

네이버 부동산에서 최신 매물 정보를 실시간으로 수집합니다. 캐시를 통해 빠른 응답을 제공합니다.

상세 필터

가격대, 면적, 매물 유형 등 다양한 조건으로 검색 결과를 필터링할 수 있습니다.

데이터 시각화

검색 결과를 차트와 그래프로 시각화하여 통계 정보를 쉽게 파악할 수 있습니다.

반응형 디자인

데스크톱, 태블릿, 모바일 등 모든 기기에서 최적화된 화면을 제공합니다.

6. FAQ (자주 묻는 질문)

Q: 크롤링이 작동하지 않아요

A: 네이버 부동산의 페이지 구조가 변경되었을 수 있습니다. 크롤링 속도 제한을 위해 지연 시간을 늘려보세요 (.env 파일에서 CRAWLER_DELAY 값 조정). 또한, Selenium을 사용해보세요 (use_selenium=True).

Q: 지도가 로드되지 않아요

A: 인터넷 연결을 확인하세요. 방화벽이 OpenStreetMap을 차단할 수 있습니다. 브라우저 개발자 도구(F12)에서 오류 메시지를 확인해주세요.

Q: 데이터가 최신이 아니에요

A: 캐시된 데이터가 오래되었을 수 있습니다. "데이터 업데이트" 버튼을 클릭하거나, 캐시 폴더를 삭제한 후 다시 검색하세요.

Q: 검색 속도가 느려요

A: 검색할 지역이 많거나, 네이버 서버 응답이 느린 경우입니다. 필터를 사용하여 검색 범위를 줄이거나, 캐시를 활용하세요.

Q: 내보내기 파일이 열리지 않아요

A: Excel이 설치되어 있는지 확인하세요. CSV 파일은 메모장으로도 열 수 있습니다. 인코딩 문제가 발생할 수 있으므로 UTF-8로 저장된 파일을 여세요.

8. 크롤링 윤리

⚠️ 중요: 이 서비스는 네이버 부동산의 데이터를 크롤링합니다. 다음 사항을 준수해주세요:

  • robots.txt 파일의 규정을 준수하세요
  • 과도한 요청을 보내지 마세요 (rate limiting)
  • 서버에 과도한 부하를 주지 않도록 적절한 지연 시간을 설정하세요
  • 수집한 데이터를 상업적으로 무단 사용하지 마세요
  • 저작권 및 이용약관을 준수하세요

9. 문제 해결

Python 패키지 설치 오류

# pip 업그레이드
pip install --upgrade pip

# 특정 패키지 설치
pip install --upgrade fastapi uvicorn

Selenium ChromeDriver 오류

# webdriver-manager 사용 (자동 설치)
pip install webdriver-manager

# Python 코드에서 사용
from webdriver_manager.chrome import ChromeDriverManager
driver = webdriver.Chrome(ChromeDriverManager().install())

포트 충돌

# 다른 포트 사용
uvicorn app.main:app --port 8001
